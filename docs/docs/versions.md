## v0.4.3

功能更新：

1. 支持非加密抄送
2. 主页显示到达率
3. 支持头像修改

## v0.4.2

功能更新：

1. 新增发送附件功能
   在新建附件中选择的附件是全局附件，每个收件箱都会收到，如果要针对个别收件箱发送不同的附件，可以在数据里面添加 attachments 字段不重写，多个文件用冒号分隔。

bug 修复：

1. 修复选择数据文件错误导致的发送失败问题

## v0.3.2
